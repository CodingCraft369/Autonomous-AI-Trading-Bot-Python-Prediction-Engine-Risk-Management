👥 Multi-User Guide

✓ How it works
• Each browser/tab stores its own API keys locally
• Keys are sent securely in request headers with each AI call
• Server never sees or stores your API keys
• Multiple users can use the same bot server simultaneously
• Each user's AI predictions use their own API key
• Trade history is shared (same bot, same account)
🔒 Security
• Keys never leave your browser except in encrypted HTTPS headers
• Server logs do NOT contain key values
• Keys are stored with browser AES encryption (standard localStorage)
• Clearing browser data completely removes all keys
• Keys are NOT synced across browsers or devices (by design)
